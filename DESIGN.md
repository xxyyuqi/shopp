# DESIGN.md

## 产品灵魂

**意象锚点：地中海海岸的阳光精品店**

想象一家开在希腊圣托里尼悬崖边的服装铺：粉刷白墙吸饱了整个夏天的阳光，门口挂着深海蓝色的亚麻布幔在午后微风里轻轻晃动，柠檬黄的遮阳棚把光线过滤成温暖的色块投在木地板上。空气里飘着亚麻、棉布与海盐的气味——这是一家"被太阳晒透的服装店"。

我们不做廉价电商模板（科技蓝+蓝紫渐变+圆角卡片堆叠+阴影），而是做**一本可以下单的时装杂志**。

## Design Tokens

### 色彩

```
--c-deep-marine:  #0B3D91   /* 深海蓝 - 骨架色（导航、主标题、强调文字） */
--c-ink:          #061F4A   /* 墨蓝 - 正文文字 */
--c-sun-yellow:   #FFD23F   /* 柠檬黄 - 唯一焦点色（CTA、价签、徽章、强调线） */
--c-sun-soft:     #FFE891   /* 柔黄 - 悬停态、副焦点 */
--c-stucco:       #FAF7F2   /* 粉刷白 - 主背景（不是 #FFFFFF，是吸饱阳光的灰泥墙） */
--c-pure-white:   #FFFFFF   /* 纯白 - 商品图底、卡片提亮 */
--c-sand:         #E8E4DC   /* 沙石灰 - 分割线、次级边框 */
--c-shadow-blue:  rgba(11,61,145,0.08) /* 海雾阴影 */
```

**色彩使用原则**：
- 黄色**只做点缀**，永远不做大面积底色（避免变 IKEA）
- 深海蓝是骨架，柠檬黄是焦点，粉刷白是空气
- 禁止使用渐变（除非是同色相极弱过渡）

### 字体

- 标题（衬线，杂志感）：`'Playfair Display', 'Noto Serif SC', Georgia, serif`
- 正文（无衬线，干净）：`'Inter', -apple-system, 'Noto Sans SC', sans-serif`
- 价格/数字：`'Playfair Display'` italic，强调编辑式价签气质

**字号节奏**：
- Display：clamp(48px, 8vw, 96px)，标题压字距 -0.02em
- H1：clamp(32px, 4vw, 56px)
- H2：clamp(24px, 3vw, 36px)
- Body：16px / line-height 1.7
- Caption：13px / letter-spacing 0.08em / uppercase

### 间距与圆角

- 间距基准：8px
- 板块间距：clamp(64px, 10vw, 128px) - 大量呼吸感
- 圆角：**最多 2px**（极简编辑感），按钮/输入框 0px，图片 0px。避免一切"圆角卡片"。

### 阴影

- 仅在悬停态使用极弱阴影：`0 24px 48px -24px var(--c-shadow-blue)`
- 静态层不要阴影，靠留白与分割线区分层级

## 视觉策略

### 摄影方向
- 商品图：4:5 竖版构图，模拟时装杂志大片
- 模特图为主，平铺图为辅
- 商品图底色为纯白 `#FFFFFF`，与页面 `#FAF7F2` 形成微妙层级
- 悬停时图片**向下偏移露出柠檬黄底衬色块**（不是 zoom，是"翻开"的感觉）

### 图形语言
- 装饰元素：极细蓝色线条（1px）+ 柠檬黄实心几何（圆点/竖条）
- 图标：1.5px stroke SVG 线条图标（lucide 风格），不用 emoji，不用填充图标
- 标题旁可点缀一条 40px 的柠檬黄竖线作为视觉锚点

## 布局与排版

- 主容器最大宽度 1440px，左右内边距 clamp(20px, 5vw, 80px)
- 首页采用**非对称编辑式网格**（如 5:7、4:8），打破对称电商感
- 标题压字距、首字下沉、行末破折号等杂志排版手法
- 大量使用 `letter-spacing: 0.1em` 的小型大写字母作为 eyebrow 标签

## 动效

- 缓动统一：`cubic-bezier(0.16, 1, 0.3, 1)`（丝绸感，慢入快出）
- 时长：260ms - 600ms（不要 150ms 这种"咔哒"感）
- 悬停：图片平移 + 黄色底衬显现，按钮反色翻转
- 入场：fade + translateY(16px)，分批触发

## 设计禁忌（不要做）

- ❌ 蓝紫色渐变 / 任何 hero 渐变
- ❌ 大圆角卡片（>4px）+ 多层阴影堆叠
- ❌ 把黄色当大面积底色（会变 IKEA / 美团）
- ❌ 饱和度过高的"科技蓝"（#1E88E5、#2196F3 这类）
- ❌ Emoji 图标 / Material 填充图标
- ❌ "促销红"、"折扣价橙" 等额外颜色破坏三色体系
- ❌ 居中对称的电商首页 Hero（保留杂志的非对称感）
- ❌ 默认 `cursor: pointer` 黑边框 outline

## 组件规范

### 按钮
- 主按钮：深海蓝底 + 白字，悬停变柠檬黄底 + 墨蓝字
- 次按钮：透明底 + 深海蓝 1.5px 描边 + 深海蓝字
- 形状方正，0 圆角，padding 18px 36px，letter-spacing 0.08em

### 价签
- 数字使用 Playfair Display italic，前面跟一个小号 ¥
- 原价用墨蓝色 line-through，现价用深海蓝或柠檬黄高亮底

### 徽章
- "HOT" / "NEW"：柠檬黄实心方块 + 墨蓝字，10px uppercase
